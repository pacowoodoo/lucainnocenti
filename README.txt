
SUPER SEO SITEMAP by Luca Innocenti

This extension simply print inside your sitemap images for product and categories
as in google standard specs. This feature will add label (that google can use in 
standard search) and img links that google will use in image search.

#Facts
version: 1.0.0
•extension key: Lucainnocenti_Superseositemap
•direct download link http://www.lucainnocenti.com/download/Lucainnocenti_Superseositemap-1.0.0.zip

#Requirements
•PHP >= 5.2.0
•Mage_Core_Sitemap

#Compatibility
•Magento >= 1.4

#Installation Instructions
1.Download the zip archive and open it.
2.Paste the "app" folder inside your magento installation.
2.Clear the cache, logout from the admin panel and then login again.
3.Configure and activate the extension under System - Configuration - Google Sitemap - "Super Seo Sitemap" tab.
4.Choose which kind of images you want that sitemap print, then regenerate your sitemap.

#Uninstallation
1.Remove all extension files from your Magento installation
